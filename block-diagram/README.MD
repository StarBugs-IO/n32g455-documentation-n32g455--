| Компонент | Описание |
| :--- | :--- |
| **Cortex-M4F** | 32-битное ядро с аппаратной поддержкой вычислений с плавающей точкой |
| **FPU** | Ускоряет математику с float/double (полезно для фильтров, FOC моторов) |

| **NVIC** | Управляет прерываниями от всех периферийных блоков |
| **8KB I-Cache** | Ускоряет выполнение кода из Flash, уменьшая количество wait-состояний |

## Уровень 3: AHB Bus Matrix
┌─────────────────────────────────────────────────────────────────────────────┐
│ AHB BUS MATRIX │
│ ┌─────────────────────────────────────────────────────────────────────┐ │
│ │ Связывает: CPU ↔ Flash ↔ SRAM ↔ DMA ↔ APB1/APB2 мосты │ │
│ │ Позволяет параллельный доступ: CPU читает Flash, DMA пишет в SRAM │ │
│ └─────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘

text

**AHB Bus Matrix** — это коммутатор, который позволяет разным устройствам одновременно обращаться к памяти. Например, CPU выполняет код из Flash, а DMA в это время пишет данные в SRAM — без задержек и конфликтов.
