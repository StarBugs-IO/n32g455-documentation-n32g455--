# 📚 Документация по микроконтроллеру N32G455

Сборник технических статей на русском языке, посвящённый микроконтроллеру **N32G455** от Nations Technologies (ARM Cortex-M4F, 144 МГц, до 512 КБ Flash, 144 КБ SRAM).

| [Полная расшифровка блок-схемы](./block-diagram/README.md) | Детальный разбор всех блоков N32G455: ядро, память, шины, периферия (UART, I2C, SPI, CAN, таймеры, ADC, DAC, COMP, OPAMP, RTC, USB). Текстовые ASCII-схемы, сводные таблицы, связи между блоками. | ✅ Готово |
