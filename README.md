# 📚 Документация по микроконтроллеру N32G455

Сборник технических статей на русском языке, посвящённый микроконтроллеру **N32G455** от Nations Technologies (ARM Cortex-M4F, 144 МГц, до 512 КБ Flash, 144 КБ SRAM).

## 🎯 Цель проекта

Предоставить понятную, структурированную и подробную документацию для разработчиков, студентов и инженеров, которые работают с N32G455. В отличие от официального даташита (сухого и сложного для новичков), здесь материал излагается доступно, с визуальными схемами, таблицами и практическими примерами.
